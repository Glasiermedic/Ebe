# Ebe Biblical Context Design

## Purpose

Ebe may optionally provide biblical context connected to a remembered situation.

This is an interpretive support layer, not a factual-memory layer.

## Separation

```text
Stored memory evidence
    ↓
Grounded factual response
    ↓
Optional biblical context
```

A biblical interpretation must not alter the underlying Memory Stone.

## Possible Modes

- Scripture reference;
- encouragement;
- prayer prompt;
- thematic analogy;
- moral reflection;
- lament;
- gratitude.

## Required Labels

The response should distinguish:

```text
Stored memory
Inference
Biblical reflection
Direct Scripture reference
```

## Safeguards

- no invented quotations;
- include book, chapter, and verse;
- identify translation when quoting;
- keep quotations within licensing constraints;
- avoid presenting one interpretation as undisputed where traditions differ;
- do not intensify paranoia, guilt, fear, or accusations;
- do not claim that God revealed an unstored factual conclusion;
- allow the feature to be disabled.

## Retrieval

A future biblical reference layer should use a curated and versioned corpus rather than relying only on model memory.
