# Ebe Engineering Documentation

This directory is the durable engineering notebook for Ebe.

## Start here

1. `roadmap/CURRENT_STATE.md` — what exists now and what is next.
2. `roadmap/ROADMAP.md` — the approved and proposed implementation sequence.
3. `roadmap/DECISIONS.md` — why the plan changed and who approved it.
4. `roadmap/MILESTONES.md` — completed capability checkpoints.
5. `roadmap/ARCHITECTURE.md` — the current system architecture.

## Documentation rule

A material architectural suggestion is not automatically part of the approved plan.

It should move through this lifecycle:

```text
Proposed
    ↓
Reviewed
    ↓
Approved or Rejected
    ↓
In Progress
    ↓
Completed, Blocked, or Superseded
```

Every completed roadmap item must identify its acceptance criteria and verification.
